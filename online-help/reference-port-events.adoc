= 端口事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
端口事件可为您提供有关集群端口的状态，以便您可以监控端口上的更改或问题，例如端口是否已关闭。



== 影响区域：可用性

|===
| 事件名称（陷阱名称） | 影响级别 | 源类型 | severity 


 a| 
端口状态为已关闭（ ocumEvtPortStatusDown ）
 a| 
意外事件
 a| 
Node
 a| 
严重

|===
