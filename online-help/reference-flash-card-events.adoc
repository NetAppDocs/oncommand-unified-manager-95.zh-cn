= 闪存卡事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
闪存卡事件可为您提供有关数据中心节点上安装的闪存卡的状态的信息，以便您可以监控潜在的问题。事件按影响区域分组，并包括事件和陷阱名称，影响级别，源类型和严重性。
