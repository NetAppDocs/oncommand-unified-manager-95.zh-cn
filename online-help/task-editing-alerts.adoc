= 编辑警报
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以编辑警报属性，例如与警报关联的资源，事件，收件人，通知选项，通知频率， 和关联脚本。



== 开始之前

您必须具有OnCommand 管理员角色。



== 步骤

. 在左侧导航窗格中、单击*配置*>*警报*。
. 在*配置/警报*页面中、选择要编辑的警报、然后单击*编辑*。
. 在 * 编辑警报 * 对话框中，编辑名称，资源，事件和操作部分， 根据需要。
+
您可以更改或删除与警报关联的脚本。

. 单击 * 保存 * 。
