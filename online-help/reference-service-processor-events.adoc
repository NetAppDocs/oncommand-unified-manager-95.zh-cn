= 服务处理器事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
服务处理器事件为您提供处理器状态的信息，以便您可以监控潜在问题。事件按影响区域分组，并包括事件和陷阱名称，影响级别，源类型和严重性。
