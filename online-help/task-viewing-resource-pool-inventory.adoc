= 查看资源池清单
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用保护/资源池页面查看资源池清单并监控每个资源池的剩余容量。



== 开始之前

您必须具有OnCommand 管理员或存储管理员角色。



== 步骤

. 在左侧导航窗格中，单击 * 保护 * > * 资源池 * 。
+
此时将显示资源池清单。
