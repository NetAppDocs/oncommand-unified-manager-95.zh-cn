= 自定义事件报告以显示任务关键型对象上的事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以自定义事件报告以显示按任务关键型数据优先级筛选的事件。



== 关于此任务

您也可以通过转到报告页面并单击相应报告的*运行报告*来执行此任务。



== 步骤

. 要筛选具有任务关键型数据优先级的事件、请单击*数据优先级*列、然后通过单击打开*筛选器*对话框 image:../media/click-to-filter.gif[""] 图标。
. 从*条件*列表中、选择*等于*。
. 单击*选择值*并选择*任务关键型*。
. 单击 * 确定 * 。
