= 自定义SVM清单报告以显示已停止SVM的列表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以自定义SVM清单报告以显示已停止SVM的列表。此报告将按状态筛选SVM。



== 关于此任务

您也可以通过转到报告页面并单击相应报告的*运行报告*来执行此任务。



== 步骤

. 要按状态筛选SVM、请单击*状态*列、然后通过单击打开*筛选器*对话框 image:../media/click-to-filter.gif[""] 图标。
. 从*条件*列表中、选择*等于*。
. 单击*选择值*并选择*已停止*。
. 单击 * 确定 * 。
