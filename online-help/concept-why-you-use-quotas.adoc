= 为什么要使用配额
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用配额限制 FlexVol 卷中的资源使用量，在资源使用量达到特定级别时提供通知或跟踪资源使用量。

指定配额的原因如下：

* 限制用户或组可以使用的磁盘空间量或文件数量，或者限制 qtree 可以包含的文件数量
* 跟踪用户，组或 qtree 使用的磁盘空间量或文件数量，而不施加限制
* 在用户的磁盘使用量或文件使用量较高时向用户发出警告
