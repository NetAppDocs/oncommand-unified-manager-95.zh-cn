= 什么是命名空间成分卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
每个无限卷都有一个命名空间成分卷、用于将目录信息和文件名映射到无限卷中文件的物理数据位置。

客户端不知道命名空间成分卷、也不会与其直接交互。命名空间成分卷是无限卷的内部组件。
