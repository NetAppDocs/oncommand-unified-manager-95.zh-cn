= 编辑资源池对话框
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用编辑资源池对话框更改现有资源池的名称和问题描述。例如，如果原始名称和问题描述不准确或不正确，您可以对其进行更改，使其更精确。



== 文本框

文本框用于更改选定资源池的以下信息：

* * 资源池名称 *
+
用于输入新名称。

* * 问题描述 *
+
用于输入新的问题描述。





== 命令按钮

命令按钮可用于执行以下任务：

* * 保存 *
+
保存对资源池名称和问题描述所做的更改。

* * 取消 *
+
丢弃所做的更改并关闭编辑资源池对话框。
