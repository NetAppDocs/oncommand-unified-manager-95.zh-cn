= 中止作业
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果作业完成时间过长、遇到过多错误或不再需要、您可以使用保护/作业页面中止作业。只有当作业的状态和类型允许时，才能中止该作业。您可以中止任何正在运行的作业。



== 开始之前

您必须具有OnCommand 管理员或存储管理员角色。



== 步骤

. 在左侧导航窗格中，单击 * 保护 * > * 作业 * 。
. 从作业列表中，选择一个作业，然后单击 * 中止 * 。
. 在确认提示符处，单击 * 是 * 以中止选定作业。
