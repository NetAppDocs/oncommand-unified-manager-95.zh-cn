= 共享报告
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过电子邮件将报告发送给一个或多个用户并与之共享。



== 步骤

. 在左侧导航窗格中、单击*报告*、然后为所需报告单击*运行报告*。
+
此时将显示报告详细信息页面。

. 单击*操作*>*共享*。
. 在*共享报告*对话框中、指定要与之共享报告的收件人的电子邮件地址。
+
您可以指定一个或多个电子邮件地址、以逗号分隔。

. 指定电子邮件的主题。
+
默认情况下、报告的名称显示为电子邮件的主题。

. 选择报告格式。
+
已选择_PDF_选项作为默认报告格式。如果选择XHTML格式、则收件人必须使用支持的Web浏览器打开通过电子邮件发送的报告。

. 单击*共享*。
