= 查看作业详细信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
启动作业后、您可以从保护/作业详细信息页面跟踪其进度、并监控关联的任务以确定是否存在可能的错误。



== 开始之前

您必须具有OnCommand 管理员或存储管理员角色。



== 步骤

. 在左侧导航窗格中，单击 * 保护 * > * 作业 * 。
. 在*保护/作业*页面中、单击*名称*列中的作业名称以显示与该作业关联的任务列表。
. 单击某个任务可在任务列表右侧的 * 任务详细信息 * 窗格和 * 任务消息 * 窗格中显示追加信息。
