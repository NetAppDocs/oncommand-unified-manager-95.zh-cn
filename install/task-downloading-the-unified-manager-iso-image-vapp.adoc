= 下载Unified Manager ISO映像
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在升级Unified Manager之前、您必须从NetApp 支持站点 下载Unified Manager ISO映像。



== 开始之前

您必须具有 NetApp 支持站点的登录凭据。



== 步骤

. 登录到NetApp 支持站点 并导航到软件下载页面。
. 下载并保存 `.iso` 将映像文件映射到vSphere Client可访问的本地目录或网络目录。
. 验证校验和以确保软件下载正确。
